# 로스트아크 특화 달력 서비스

로스트아크 관련 일정을 달력 형태로 확인하고, 카카오톡 연동을 통해 개인 일정과 알림 기능까지 사용할 수 있는 웹 서비스입니다.

현재는 로스트아크만 대상으로 개발하며, 로그인 기능은 만들지 않습니다.

---

## 프로젝트 개요

이 프로젝트의 핵심은 **달력**입니다.

로스트아크의 주요 일정과 사용자가 직접 등록한 일정을 하나의 달력에서 관리하고, 카카오톡 연동 사용자는 일정에 대한 알림을 받을 수 있습니다.

---

## 핵심 컨셉

- 로그인 없는 로스트아크 특화 달력 서비스
- 모든 사용자는 로아 달력 조회 가능
- 카카오톡 연동자는 일정 등록 및 알림 사용 가능
- 알림은 독립 기능이 아니라 일정에 연결되는 기능
- 로아 일정은 시스템 일정으로 제공
- 사용자 일정과 로아 일정을 하나의 달력에 함께 표시
- 초기 수익화는 고려하지 않으며, 추후 후원/광고/그룹 기능 중심으로 검토

---

## 사용자 유형

| 사용자 유형   | 설명                                                             |
| ------------- | ---------------------------------------------------------------- |
| 미연동 사용자 | 카카오톡 연동 없이 로아 달력을 조회하는 사용자                   |
| 개인 연동자   | 카카오톡 1대1 연동 후 개인 일정과 알림을 사용하는 사용자         |
| 그룹 사용자   | 카카오톡 단체방에 봇을 연동해 그룹 일정과 알림을 사용하는 사용자 |

## 주요 기능

### 1. 로아 달력 조회

- 모험섬
- 카오스게이트
- 필드보스
- 공지
- 패치노트
- 점검
- 이벤트
- 이벤트 종료일
- 상점/패키지

로아 시스템 일정은 사용자마다 복사 저장하지 않고, 공통 데이터로 관리합니다.

---

### 2. 사용자 일정

카카오톡 연동자는 직접 일정을 등록할 수 있습니다.

일정 항목:

- 제목
- 시간
- 반복
- 내용
- 색상

미연동 사용자는 직접 일정을 등록할 수 없습니다.

---

### 3. 알림

달력에 표시되는 모든 항목은 알림 등록 대상이 될 수 있습니다.

- 사용자 일정 알림
- 로아 시스템 일정 알림
- 카카오톡 1대1 알림
- 단체톡 알림

알림은 카카오톡 연동자만 사용할 수 있습니다.

---

### 4. 카카오톡 연동

로그인 기능 대신 카카오톡 연동으로 사용자를 식별합니다.

개인 연동:

- 카카오톡 사용자 식별값 저장
- 개인 일정 사용
- 개인 알림 사용
- 1대1 봇 기능 사용

그룹 연동:

- 카카오톡 단체방 식별값 저장
- 그룹 일정 사용
- 단체톡 알림 사용
- 그룹 봇 기능 사용

---

### 5. 봇

봇은 비즈니스 로직을 최소화합니다.

처리 흐름:

```text
카카오톡 메시지 수신
→ 명령어 판별
→ 서버 API 호출
→ 서버 응답 그대로 답장
```
