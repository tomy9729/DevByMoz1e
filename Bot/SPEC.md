# 봇 명령어 명세

## 기준 문서

- 참고 사이트: https://kbotdocs.dev/
- 봇 코드는 `bot/main`에서 서버 봇 전용 API를 호출하고, 서버 응답 문자열을 그대로 답장한다.
- 명령어 추가 시 `bot/main`, 봇 전용 서버 API, `!명령어` 안내 문구를 함께 반영한다.

## 메신저봇R JS 문법 기준

- 현재 저장소에서 실제 메신저봇R 앱 버전 정보는 확인되지 않는다.
- 현재 코드는 KBotDocs API2의 `BotManager.getCurrentBot()`, `bot.setCommandPrefix()`, `Event.COMMAND`, `Command.command`, `Command.args`, `Command.reply()` 기준으로 작성한다.
- 실제 구현은 현재 사용 중인 메신저봇R 버전 기준으로 동작해야 한다.
- 허용 가능한 JS 문법 범위 내에서만 구현한다.
- 최신 JS 문법 사용 가능 여부가 불명확하면 보수적으로 작성한다.

## 명령어 추가 기준

- 표시용 명령어명, 별칭 목록, 설명, 연결 API, 파라미터 파싱 규칙을 함께 관리한다.
- 구현 완료된 명령어만 `!명령어` 응답에 노출한다.
- 단순 조회형 명령어를 먼저 추가하고, 알람/개인설정형 명령어는 별도 단계로 확장한다.
